# MO.RPG Bot

A community-driven Discord bot for the Royal Conquest Alliance.

## What is this?

**MO.RPG** is a fan-made MMORPG-style Discord bot inspired by Supercell’s **mo.co**.
The goal is to recreate a **mo.co-like experience and atmosphere** while adapting gameplay for a Discord-based community.

This project is built **by and for the community** — contributions, ideas, and discussion are all welcome.

## Project Status

This project is in **early development**.
Most features listed below are **planned**, not yet fully implemented.

## Planned Features

- **Gear System**  
  Unlock **Passives**, **Gadgets**, **Weapons**, and **Smart Rings**

- **Leveling**  
  Level up your character to increase your **power level**

- **Rifts**  
  Cooperative **multiplayer boss fights** with up to **4 players**

- **PvP**  
  Competitive **1v1 battles** to climb the ranks

- **Modifier System**  
  Each piece of gear feels unique and impactful
